Repaso de HTML

1. Listas y Enlaces

  • <li>: Define cada elemento de una lista (ordenada o desordenada).
  • <ul>: Crea listas desordenadas (sin orden específico).
  • <ol>: Crea listas ordenadas (alfabéticas o numéricas).
  • <a>: Define enlaces para otras páginas, archivos o rutas internas.
  • href: Vincula páginas, motores de búsqueda o partes internas de la misma página.

2. Formularios

  • <label>: Asocia etiquetas con campos de entrada.
  • <select>: Crea menús desplegables.
  • <button>: Elemento de botón clickeable.
  • <form>: Representa un formulario que envía información al servidor.
  • <input>: Crea controles interactivos para recibir datos.
  • <textarea>: Permite edición multilínea de texto sin formato.

3. Etiquetas de Texto y Encabezados

  • <h1> a <h6>: Títulos y subtítulos jerárquicos.
  • <hr>: Separador de párrafos; indica cambio de tema.
  • <strong>: Indica importancia, seriedad o urgencia.
  • <em>: Enfatiza el contenido.
  • <br>: Inserta saltos de línea.
  • <p>: Define párrafos de texto e imágenes.

4. Tablas

  • rowspan: Extiende una celda verticalmente a varias filas.
  • colspan: Extiende una celda horizontalmente a varias columnas.
  • <th>: Define celdas de encabezado.
  • <table>: Crea la tabla principal.
  • <td>: Define celdas estándar de datos.
  • <tr>: Representa una fila en la tabla.

5. ¿Qué es HTML?

HTML (HyperText Markup Language) es el lenguaje estándar para estructurar el contenido de páginas web, utilizando etiquetas para definir elementos.

6. Estructura Básica de un Documento HTML

  • <title>: Contiene el título del documento.
  • Doctype: Define el tipo de documento y su sintaxis.
  • <body>: Incluye todo el contenido visible de la página.
  • <html>: Elemento raíz que abarca todo el documento.
  • Comentarios: Anotaciones en el código para aclarar secciones.
  • <head>: Contiene metadatos (información general no visible).

7. Elementos Multimedia

  • <img>: Inserta imágenes en la página web.
  • <video>: Inserta videos; permite definir la fuente del video.
  • Formatos de imagen: JPG, GIF, PNG, SVG, WEBP.
  • Formatos de video: MP4, OGG, WEBM.

8. Historia de HTML

Las versiones estándar antes de HTML5 fueron: HTML 2.0, HTML 3.2 y HTML 4.01.
